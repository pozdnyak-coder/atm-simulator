# 🏧 ATM Simulator — Симулятор Банкомата

Веб-приложение банкомата на Python (Flask).

## Функции
- 🔐 Вход по номеру карты + PIN-код (3 попытки, затем блокировка)
- 💰 Просмотр баланса
- 💸 Снятие наличных (кратно 1000 сум)
- 💳 Внесение наличных
- 📋 История последних 20 операций

## Демо-карты
| Карта | PIN  | Баланс       |
|-------|------|-------------|
| 1234  | 1111 | 150 000 сум |
| 5678  | 2222 |  75 000 сум |
| 9999  | 0000 | 500 000 сум |

## Запуск локально
```bash
pip install -r requirements.txt
python app.py
```
Открыть: http://localhost:5000

## Технологии
- Python 3.11
- Flask 3.0
- Gunicorn (для деплоя)
- HTML/CSS (без сторонних библиотек)
